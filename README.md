## 📁 폴더 구조
```
📦 data-contest-lab
 ┣ 📁 meetings/              ← 회의록 (스터디 회의록, 일정별 정리)
 ┣ 📁 notebooks/             ← 실습용 주피터 노트북, 코드 실험 결과
 ┣ 📁 datasets/              ← 데이터셋 (대용량 파일은 .gitignore 처리)
 ┣ 📁 models/                ← 모델 학습 및 평가 코드
 ┣ 📁 submissions/           ← 공모전 제출 코드 / 결과물
 ┣ 📁 references/            ← 참고 자료 (논문, 링크 등)
 ┣ 📁 docs/                  ← 보고서, 발표자료, 발표 슬라이드
 ┣ 📄 README.md
 ┗ 📄 requirements.txt
```
